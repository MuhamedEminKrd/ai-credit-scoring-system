# 🏦 AI-Powered Credit Scoring System

An end-to-end Machine Learning pipeline designed to evaluate customer credit risk and automate loan approval processes. This project demonstrates the application of classification algorithms to solve real-world financial risk management challenges.

## 🚀 Key Engineering Features

* **Synthetic Data Engineering:** Engineered a highly realistic, statistically sound dataset of 1,000 banking customers using `NumPy`, bypassing strict KVKK (data privacy) limitations while maintaining analytical integrity.
* **Exploratory Data Analysis (EDA):** Visualized feature correlations (e.g., Annual Income vs. Existing Debt) using `Seaborn` to map the risk landscape before model training.
* **Machine Learning Architecture:** * Implemented a **Random Forest Classifier** (`scikit-learn`) to predict loan approval/rejection.
    * Split data into Training (80%) and Testing (20%) sets to validate model accuracy and prevent overfitting.
* **Automated Inference:** Developed an automated decision engine that evaluates new, unseen customer profiles in milliseconds, returning actionable business decisions ("Approved" or "Rejected" with risk context).
